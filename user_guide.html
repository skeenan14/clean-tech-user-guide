User Guide for the Clean Technology Data Dashboard

Introduction

This interactive dashboard allows you to explore clean technology data across different countries and years.

Using the Dashboard

Use the controls on the left to customize the visualization:

  • Select Country: Choose a country from the dropdown menu.
  • Select Energy Source: Check the boxes for the energy sources you want to visualize.
  • Select Year Range: Adjust the sliders to set the range of years.
  • Select Plot Type: Choose between a line plot and a bar plot.
  • Add Annotation: Enter any text to annotate the plot.
  • Show Data Table: Check the box to display the underlying data table.
